# demo-idea

## Hypothesis (rough)

远程工作者经常错过会议，因为他们所在的时区与会议组织者不同，现有日历工具没有智能时区提醒功能。我的解决方案是一个跨时区智能日历助手，自动检测参与者的时区差异，在会议前发送个性化提醒，并建议替代时间。

## Why I think this

我自己在远程团队工作了 5 年，每周至少错过 1-2 次会议。同事也频繁抱怨同样的问题。Slack 上 "sorry I missed the meeting" 是最常见的消息之一。

## Who I think has this problem

- 跨国公司的软件工程师（50-5000 人规模）
- 分布式团队的 PM/项目经理
- 自由职业者同时对接多个时区的客户
